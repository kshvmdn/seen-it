# seen-it
Stream YouTube content from /r/videos. Replacement for the now deprecated reddit.tv.

#### Usage

```
git clone http://github.com/kshvmdn/seen-it
```

```
bower install
```

```
grunt serve
```

Open `localhost:PORT` (check the console for `PORT`) in your browser.
